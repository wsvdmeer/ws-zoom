ws-carousel
===========

Minimal jquery carousel

How to use:

Html:
```html
<div class="zoom">
				<img src="images/thumb.jpg" data-large="images/large.jpg"/>
</div>
```

Jquery:
```html
$(".zoom").wszoom();
```

Defaults:
<ul>
<li>windowXpos: 0</li>
<li>windowYpos:0</li>
<li>windowWidth:300</li>
<li>windowHeight:300</li>
</ul>
